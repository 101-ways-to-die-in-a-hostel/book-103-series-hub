## Artist Instructions

### Do:
- Emphasize the horror-comedy tone (dark but not gratuitous)
- Make the death method recognizable but intriguing
- Create shelf appeal that stands out
- Ensure series consistency with other covers
- Include subtle @HostelReaper element

### Don't:
- Show explicit gore or violence
- Use cliché horror imagery (screaming faces, etc.)
- Make it look like generic thriller
- Neglect the dark comedy element
- Forget the book number badge

### Technical Requirements
- Resolution: 300 DPI minimum
- Color mode: CMYK
- Bleed: 0.125" on all sides
- Safe zone: 0.25" from trim on all sides
- Spine: Calculate based on [X] pages, [paper weight]
- File format: PDF/X-1a:2001 (for print), PNG (for digital)
